# #english-note 
# solving equations
$$\begin{align}
ax + by = u ,\\
cx + dy = v.
\end{align}$$
$$\begin{align}
c(ax+by)=cu\\
a(cx+dy)=av
\end{align}$$
$$cax+cby-(acx+ady)=cu-av$$
$$bcy-ady=cu-av$$
$$(bc-ad)y=cu-av$$
$$y=\frac{cu-av}{bc-ad}$$
$$\begin{align}
d(ax+by)=du\\
b(cx+dy)=bv
\end{align}$$
$$dax+dby-(bcx+bdy)=du-bv$$
$$dax-bcx=du-bv$$
$$(ad-bc)x = du-bv$$
$$x=\frac{du-bv}{ad-bc}=\frac{bv-du}{bc-ad}$$
